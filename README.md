# 📚 Novella - Social Platform for Writers & Readers

Novella is a modern, interactive social platform designed specifically to bridge the gap between creative writers and enthusiastic readers. It’s not just a reading app; it’s a community where stories come to life through interaction.

## ✨ Key Features

-   **Dual-Role System:** Distinct experiences for "Authors" (content creators) and "Readers" (audience).
-   **Interactive Community:** Explore other users, build a network of friends, and follow your favorite authors.
-   **Private Library:** A dedicated space for users to save their favorite stories and manage their following list.
-   **Real-time Interactions:** Support for Likes, Comments, and adding novellas to personal Favorites.
-   **Secure Authentication:** Robust login system using JWT to ensure a safe environment for all users.

## 🛠️ Tech Stack

### Frontend
-   **React.js & TypeScript:** For a fast, type-safe, and interactive User Interface.
-   **Tailwind CSS:** For modern, responsive, and sleek styling.
-   **Shadcn UI & Lucide Icons:** For professional and consistent UI components.

### Backend
-   **Node.js & Express.js:** Powering the server-side logic and API endpoints.
-   **Prisma ORM:** Advanced database management and seamless data relations.
-   **PostgreSQL:** Reliable and scalable relational database.
-   **JWT (JSON Web Tokens):** Secure user authentication and authorization.
